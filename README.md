## Ferramentas
- HTML
- CSS
- ReactJS

## Instalação
- Primeiro clone o repositório :
`$ git clone https://github.com/IggorSantos/grindhouse.git`
- Depois entre na pasta do projeto : `$ cd grindhouse`
- Instale as dependencias: `$ yarn install`
